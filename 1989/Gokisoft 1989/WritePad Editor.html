WritePad Editor

WritePad Editor is an editor application that allows you to create Web pages easily and quickly.

Steps to open WritePad application:

  • Click Start
  • Click Programs
  • Click WritePad


Advantages of using a WritePad editor:
  1. It allows faster creation of Web pages.
  2. It allows easier updation of text or images in WritePad editor.
  3. It is platform independent


Disadvantages of using a WritePad editor:
  1. It consumes more memory space.
  2. It requires to fill-in more details such as meta data for each page.
